# Formatting
## Word Spacing
> [!info]+ Single Spaced Only
> - *Accommodation:* **Userscript written by Ziggy**
> 
> I rely *heavily* on a userscript I wrote for myself that ensures any and all text is displayed as single text, removing almost all additional spacing anyone might add for their "aesthetic". This userscript is the *only way* I can actually interact with anyone who chooses to use any type of additional spacing for their posts on tumblr.
## Font Size
> [!info]+ Single Spaced Only
> - *Accommodation:* **Stylesheets written by Ziggy**
> 
> I rely on a pair of stylesheets to handle font sizing on tumblr — one to change the font entirely to one of the more easily read alternatives in the font family and make the default size smaller, and another to handle adjusting the size of small (and any double small) text to be larger than it is by default but not so much you can't tell a difference between it and normal (which isn't so jarring of a difference as it is on the web otherwise, without these stylesheets).
